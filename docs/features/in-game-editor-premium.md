# ✏️In-game Editor - Premium

* Download the addon plugin [here](https://www.spigotmc.org/resources/ultimateshopeditor-in-game-shop-editor-for-ultimateshop-plugin-w-i-p.99542/).
* Restart your server. This addon plugin require use UltimateShop premium 2.2.0+.
* Use `/shop editor` in server, this command require `ultimateshop.editor` permission.
* Because this system does not finish at the moment, so it will display a warning message, read it then use `/shop editor` twice to open the editor. You just need do this once after join the server.
* For now, you can only create or edit shop, and create or edit product in editor. Also, the editor for now can only edit price and sub product in product editor.
